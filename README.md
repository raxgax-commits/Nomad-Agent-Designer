# Nomad Agent Designer

A design and orchestration tool for building, configuring, and deploying autonomous AI agents.

## Overview

Nomad Agent Designer provides a framework for creating intelligent agents that can plan, reason, and execute tasks autonomously. Design agent workflows visually, configure capabilities, and deploy them to production.

## Getting Started

### Prerequisites

- Node.js >= 18
- npm >= 9

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

```
src/
  core/       — Agent runtime and orchestration engine
  agents/     — Agent definitions and templates
  tools/      — Tool integrations agents can use
  config/     — Configuration and schema definitions
  utils/      — Shared utilities
```
